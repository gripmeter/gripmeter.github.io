How to calibrate GripMeter device

Como calibar un dispositivo GripMeter

Open the application in the normal way.
Abrir aplicación de forma normal.

1- Log in.
If it is the first time it will be necessary to select the email and accept the conditions of use of the application.
The next times the login will be automatic (until you log out).
1- Hacer login. Si es la primera vez será necesario seleccionar el email y aceptar las condiciones de uso de la aplicación. Las próximas veces el login será automático (hasta que se haga log out).

2- In the main menu enter the CONNECT option.
2- En el menú principal entrar en la opción CONNECT.

3- Make sure that the phone's Bluetooth is activated. The first time the application will ask for location permissions, it is necessary to accept the request.
The device will be listed on the screen.
3- Asegurarse que el bluetooth del teléfono esta activado. La primera vez la applicación pedirà permisos de localización, es necesario acceptar la petición.
El dispositivo saldrá listado en la pantalla.

4- Tap on the gear icon to enter the configuration menu.
4- Hacer tap en el icono de engranaje para entrar en el menú de configuración.

5- Place the device for calibration.
5- Colocar el dispositivo para la calibración.

6- Before placing the calibration weight it is necessary to tap on "TARE CURRENT" to zero the device.
6- Antes de colocar el peso de calibración es necesario hacer tap en "TARE CURRENT" para poner a cero el dispositivo.

7- Make sure that the device shows 0 kg aprox on the display.
7- Asegurarse que el dispositivo marca más o menos 0 kg en la pantalla.

8- Place the known weight on the device. It is recommended to use a weight value similar to the one you intend to measure, never more than 150 kg.

8- Colocar el peso conocido en el dispositivo. Se recomienda usar un valor de peso similar al que se pretende medir nunca superior a 150 kg.

9- Enter the value of the known weight in the input box.
9- Introducir el valor del peso conocido en el input box.

10- Tap the CALIBRATE button.
10- Hacer tap en el botón CALIBRATE

11- Make sure that the device shows the correct weight on the display. If this is not the case repeat the whole calibration process.
11- Asegurar que el dispositivo marca más o menos el peso conocido en la pantalla. Si no es el caso repetir todo el proceso de calibración.
